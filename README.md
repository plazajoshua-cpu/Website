# joshua-plaza

Personal portfolio for Joshua Plaza — Revenue and AI Marketing Operations strategist.

Built on a 3D Next.js template using React Three Fiber, GSAP, and Zustand.

## Tech Stack

- Next.js
- React
- React-three-fiber
- DREI
- GSAP
- Zustand
- Tailwind

## Getting Started

```bash
npm install
npm run dev
```
